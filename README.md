## Getting Started / 시작하기

### Prerequisites / 필요한 프로그램

```
Node.js
```

### How to start the server / 서버를 시작하는 방법

1. Install dependencies
```sh
npm install
```

2. Write .env file
```
MYSQL_ROOT_PASSWORD=

DB_PORT=blog
DB_USER=
DB_PASSWORD=

NODE_ENV=production
PORT=3000

DB_PORT=3306
DB_USER=
DB_PASSWORD=
DB_DATABASE=

SESSION_SECRET=
```

3. Start server
```sh
npm start
```
